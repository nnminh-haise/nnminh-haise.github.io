        C++ string - Chuỗi trong ngôn ngữ lập trình C++

        Làm sao để biểu diễn một tập hợp gồm các kí tự trong bảng chữ cái?

        C++ char - kiểu dữ liệu kí tự trong C++

        C++ char - kiểu dữ liệu kí tự trong C++ là một trong các kiểu dữ liệu nguyên thủy (primitive data type) của ngôn ngữ C++. Mỗi biến kiểu charsẽ lưu trữ một kí tự trong bảng mã ASCII.

        Sử dụng char trong C++

        Khai báo và khởi gán

        Ta thực hiện khai báo một biến kiểu char tương tự với cách khai báo các kiểu dữ liệu nguyên thủy khác trong ngôn ngữ C++:

        Cậu có thể đọc thêm về bảng mã ASCII ở mục tài liệu tham khảo bên dưới.

        char <tên biến>;
        1

        Ví dụ:

        char chr;
        1

        Bên cạnh việc khai báo như trên, ta cũng có thể khởi gán một biến char với một kí tự bất kì trong bảng mã ASCII:

        char <tên biến> = <kí tự bất kì>;
        1

        Ví dụ:

        char chr = 'A';
        1

        Lưu ý: kí tự trong C++ phải được bao quanh bởi dấu nháy đơn.

        Ép kiểu với char

        Giả sử nếu ta gán cho một biến kiểu int kí tự a thì biến đó sẽ có giá trị gì?

        int a = 'a';
cout << "Value of a: " << a << endl;
        1
        2

        Kết quả đầu ra sẽ là:

        Value of a: 97
        1

        Vì sao lại in ra số 979797? - Nếu bạn tra thứ tự của kí tự a trong bảng mã ASCII sẽ ra kết quả là 979797. Quy ước rằng trình dịch khi ép kiểu từ char sang int sẽ thay kí tự đó bằng thứ tự tương ứng trong bảng mã ASCII. Tương tự cho kiểu double và float. Riêng với bool thì trình dịch sẽ làm tương tự. Nhưng theo quy ước, các giá trị khác 000 sẽ là true và 000 sẽ là false nên các biến kiểu char khi ép kiểu sang bool sẽ hiểu là 111 hoặc 000.

        Mặt khác, ta cũng có thể ép kiểu số nguyên sang kiểu kí tự nhưng có một số các kí tự đặc biệt ta không quan sát được nên cần phải cẩn thận khi ép kiểu từ int sang bool.

        Một ứng dụng cho việc ép kiểu này, ta có thể in ra danh sách các chữ cái dễ dàng như sau:

        for (int i = 0; i < 26; ++i) {
    cout << (char)(i + 'a') << " ";
}
cout << "\n";

        Giải thích: bảng chữ cái tiếng anh có 262626 kí tự và kí tự đầu tiên là a. Do đó ta có một vòng lặp for lặp lại với biến iii có giá trị trong đoạn [0,25][0, 25][0,25]. Như đã đề cập ở trên, trình dịch khi ép kiểu các kí tự char sang int sẽ thay thế thứ tự của kí tự tương ứng trong bảng mã ASCII. Vì vậy biểu thức (i+‘a‘)(i + `a`)(i+‘a‘) sẽ tương đương với biểu thức (i+97)(i + 97)(i+97) và khi iii có giá trị trong đoạn [0,25][0, 25][0,25] thì biểu thức cũng có giá trị trong đoạn [97,122][97, 122][97,122]. Cuối cùng, với mỗi giá trị của biểu thức ta sẽ ép kiểu từ int sang char và thu được kí tự tương ứng (với lối tư duy tương tự, ta có thể thực hiện với các kí tự in hoa và kí tự chữ số).

        Bên cạnh đó, ta cũng có thể ép kiểu các kí tự trong bảng chữ cái tiếng anh sang thứ tự từ 000 đến 252525 như sau:

        for (char i = 'a'; i <= 'z'; ++i) {
    cout << i - 'a' << " ";
}
cout << endl;

        C-Style string - Chuỗi kiểu C

        Chuỗi kiểu C xuất hiện từ ngôn ngữ lập trình C. Cụ thể ta sẽ kết hợp mảng với kiểu dữ liệu char. Trong đó mỗi phần tử của mảng là một kí tự char và phần tử cuối cùng của mảng sẽ là kí tự null.

        Khai báo và khởi gán

        Tương tự như mảng số nguyên hay số thực, ta cũng có thể khai báo C-Style string như sau:

        char <tên biến>[<số lượng phần tử>];
        1

        Ví dụ:

        char id[20]; // Đây là mảng chứa tối đa 20 kí tự.
        1

        Hoặc ta có thể khởi gán như sau:

        char id[20] = "2A5F9C";
cout << id << endl;
        1
        2

        Output của chương trình:

        2A5F9C
        1

        Tuy khai báo một chuỗi có tối đa 202020 kí tự nhưng cậu không nhất thiết phải chứa hết 202020 kí tự đó.

        Một số hàm thông dụng đối với C-Style string

        Hàm strcpy: Sao chép chuỗi nguồn vào chuỗi đích (kể cả kí tự cuối).

        char* strcpy (char* destination, const char* source);
        1

        Ví dụ:

        #include <iostream>
#include <cstring>

using namespace std;

int main() {
    char a[20] = "1A2B3C";
    char b[20];
    cout << "Value of a: \"" << a << "\"" << endl;
    cout << "Value of b: \"" << b << "\"" << endl;
    strcpy(b, a);
    cout << "Value of a: \"" << a << "\"" << endl;
    cout << "Value of b: \"" << b << "\"" << endl;
    return 0;
}

        Hàm strcat: Hàm sẽ sao chép chuỗi nguồn vào cuối chuỗi đích. Cụ thể, kí tự null của chuỗi đích sẽ bị thay thế bởi kí tự đầu tiên của chuỗi nguồn và một kí tự null sẽ được thêm vào cuối chuỗi đích. Lưu ý: hai chuỗi nguồn và đích không nên bị chồng lên nhau.

        char* strcat (char* destination, const char* source);
        1

        Ví dụ:

using namespace std;

int main() {

    strcat(b, a);
    cout << "Value of a: \"" << a << "\"" << endl;
    cout << "Value of b: \"" << b << "\"" << endl;
    return 0;
}

        Hàm strcmp: Hàm sẽ so sánh chuỗi str1str1str1 với str2str2str2 theo quy tắc: chương trình sẽ bắt đầu so sánh từ kí tự đầu tiên của hai chuỗi, nếu hai kí tự giống nhau sẽ tiếp tục so sánh. Cho đến khi gặp hai kí tự khác nhau thì sẽ dừng và trả kết quả tương ứng.

        Hàm sẽ trả về giá trị xxx tương ứng:

        • x=0x = 0x=0 nếu hai chuỗi bằng nhau.
        • x<0x < 0x<0 nếu chuỗi str1str1str1 bé hơn chuỗi str2str2str2.
        • x>0x > 0x>0 nếu chuỗi str1str1str1 lớn hơn chuỗi str2str2str2.
        int strcmp (const char* str1, const char* str2);
        1

        Ví dụ:

    cout << "Value of a: \"" << a << "\"" << endl;
    cout << "Value of b: \"" << b << "\"" << endl;
    int result = strcmp(a, b);
    cout << "Value of result: " << result << endl;
    return 0;
}

        Hàm strlen: Hàm sẽ trả về độ dài của chuỗi strstrstr. Lưu ý: Độ dài của một C-Style string là số lượng kí tự tính từ kí tự đầu tiên đến kí tự null (không tính kí tự null). Do vậy không nên hiểu nhầm giữa độ dài chuỗi và kích thước mảng chứa các kí tự.

        size_t strlen (const char* str);
        1

        Ví dụ:

        #include <iostream>
#include <cstring>

using namespace std;

int main() {
    char a[20] = "1A2B3C";
    cout << "Value of a: \"" << a << "\"" << endl;
    size_t len = strlen(a);
    cout << "Length of a: " << len << endl;
    cout << "Size of a: " << sizeof(a) << endl;
    return 0;
}

        C++ string - string trong C++

        Basic string (hay string class - lớp string) là đối tượng (object) chứa một tập hợp các kí tự liên tiếp.

        Khai báo và khởi gán

        Ta có thể khai báo một biến kiểu string như sau:

        string <tên biến>;
        1

        Ví dụ:

        string name;
        1

        Hoặc khởi gán một giá trị cho chuỗi như sau:

        string <tên biến> = <chuỗi khởi gán>;
        1

        Ví dụ:

        string name = "James";
cout << "Value of name: \"" << name << "\"" << endl;
        1
        2

        Output của chương trình:

        Value of name: "James"
        1

        Sâu hơn về std::string

        Vì được xây dựng với cấu trúc tương tự như một cấu trúc lưu trữ tiêu chuẩn (a standard container) của C++ nên std::string có các hàm thành viên (member function) như sau:

        • (constructor) (opens new window).
        • (destructor) (opens new window).
        • operator= (opens new window).
        • Các hàm liên quan đến con trỏ.
        • Các hàm liên quan đến kích thước.
        • Các hàm liên quan đến việc truy cập đến các phần tử.
        • Các hàm biến đổi std::string.
        • Các hàm liên quan đến các phép tính std::string.

        Mình sẽ chỉ liệt kê một số hàm thông dụng, cậu có thể tìm hiểu thêm ở phần tài liệu tham khảo ở cuối.

        Toán tử truy cập phần tử (operator[]): Toán tử sẽ trả về một tham chiếu (reference) đến kí tự pospospos của chuỗi. Lưu ý: nếu vị trí pospospos bằng với độ dài chuỗi thì toán tử sẽ trả về kí tự null.

        char& operator[] (size_t pos);
const char& operator[] (size_t pos) const;

int main() {
    string name = "James";
    for (int i = 0; i < name.length(); ++i) {
        cout << name[i] << endl;

        Hàm empty(): Hàm sẽ cho biết nếu chuỗi đang xét rỗng hoặc không (rỗng nếu độ dài chuối bằng 000).

        bool empty() const noexcept;

    cout << "Value of is_empty: " << is_empty << endl;
    name = "";
    is_empty = name.empty();
    cout << "Value of is_empty: " << is_empty << endl;
    return 0;
}

        Hàm copy(): Hàm sẽ sao chép một chuỗi con có độ dài lenlenlen bắt đầu từ kí tự ở vị trí pospospos của chuỗi hiện tại vào chuỗi (c-style string) sss. Lưu ý: Chuỗi được gán (hay chuỗi chứa chuỗi con) sẽ không có kí tự null ở cuối.

        size_t copy (char* s, size_t len, size_t pos = 0) const;
        1

        Ví dụ:

        #include <iostream>
#include <string>

using namespace std;

int main() {
    char a[10];
    string name = "James";
    size_t len = name.copy(a, 2, 0);
    a[len] = '\0';
    cout << "Value of a: \"" << a << "\"" << endl;
    return 0;
}
