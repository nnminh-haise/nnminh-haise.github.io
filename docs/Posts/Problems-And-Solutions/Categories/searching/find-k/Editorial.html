    Tìm k | Editorial - Lời giải

    Tags: searching

    Statement - Đề bài

    Cho một dãy AAA có n(1≤n≤104)n \; (1 \le n \le 10^4)n(1≤n≤10​4​​) phần tử là các số nguyên dương a1,a2,a3,…,an(1≤ai≤104)a_1, a_2, a_3, \ldots, a_n \; (1 \le a_i \le 10^4)a​1​​,a​2​​,a​3​​,…,a​n​​(1≤a​i​​≤10​4​​) và một số nguyên dương k(1≤k≤104)k \; (1 \le k \le 10^4)k(1≤k≤10​4​​).

    Yêu cầu: In ra vị trí nhỏ nhất của kkk trong dãy AAA. Nếu không tìm thấy kkk trong AAA, in −1-1−1.

    Input - Dữ liệu vào

    • Dòng đầu tiên là gồm hai số nguyên dương nnn và kkk - số lượng phần tử của dãy và số cần tìm.
    • Dòng thứ hai là nnn số nguyên a1,a2,a3,…,ana_1, a_2, a_3, \ldots, a_na​1​​,a​2​​,a​3​​,…,a​n​​.

    Output - Dữ liệu ra

    • Một số nguyên duy nhất là vị trí của kkk trong dãy AAA.

    Solution - Giải thuật

    C++:

    #include <bits/stdc++.h>

using namespace std;

int main() {
    int n, k;
    cin >> n >> k;
    int *a = new int [n];
    for (int i = 0; i < n; ++i) {
        cin >> a[i];
    }

    int ans = -1;
    for (int i = 0; i < n; ++i) {
        if (a[i] == k) {
            ans = i;
            break;
        }
    }
    cout << ans << endl;

    return 0;
}
