Không xuất hiện | Editorial - Lời giải

Tags: Sorting, Distribution

Statement - Đề bài

Cho một dãy AAA gồm n(1≤n≤30000)n \; (1 \le n \le 30000)n(1≤n≤30000) số nguyên a1,a2,a3,…,an(2≤ai≤106)a_1, a_2, a_3, \ldots, a_n \; (2 \le a_i \le 10^6)a​1​​,a​2​​,a​3​​,…,a​n​​(2≤a​i​​≤10​6​​).

Yêu cầu: Tìm số tự nhiên nhỏ nhất không xuất hiện trong dãy.

Input - Dữ liệu vào

  • Dòng đầu tiên là số nguyên dương nnn.
  • Dòng thứ hai là nnn số nguyên dương của dãy AAA.

Output - Dữ liệu ra

  • Một số nguyên dương kkk duy nhất thỏa mãn yêu cầu đề bài.

Solution - Giải thuật

Ta có nhận xét sau: số tự nhiên nhỏ nhất không xuất hiện trong dãy sẽ nằm trong đoạn [0,n][0, n][0,n]. Do ñó, ta sử dụng mảng c[]c[]c[] với c[i]c[i]c[i] là số lần xuất hiện của xxx trong dãy, nếu c[x]=0c[x] = 0c[x]=0 tức là xxx không xuất hiện trong dãy.

Optimized solution - Giải thuật tối ưu

C++:

#include <bits/stdc++.h>

using namespace std;

int main() {
    int n;
    cin >> n;
    vector < int > a(n);
    for (int &item: a) {
        cin >> item;
    }

    int *f = new int [1000001];
    for (int i = 0; i <= 1000000; ++i) {
        f[i] = 0;
    }

    for (int item: a) ++f[item];
    for (int i = 0; i <= 1000000; ++i) {
        if (f[i] == 0) {
            cout << i << endl;
            return 0;
        }
    }

    return 0;
}
