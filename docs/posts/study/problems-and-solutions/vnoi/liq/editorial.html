    Dãy con tăng đơn điệu dài nhất - LIQ | Editorial - Lời giải

    Tags: Dynamic programming, basic

    Statement - Đề bài

    Nguồn: LIQ | VNOI (opens new window).

    Cho một dãy số nguyên AAA gồm n(1≤n≤1000)n \; (1 \le n \le 1000)n(1≤n≤1000) phần tử a1,a2,a3,…,an(1≤ai≤10000)a_1, a_2, a_3, \ldots, a_n \; (1 \le a_i \le 10000)a​1​​,a​2​​,a​3​​,…,a​n​​(1≤a​i​​≤10000).

    Biết dãy con tăng đơn điệu là 111 dãy ai1,ai2,ai3,…,aika_{i_1}, a_{i_2}, a_{i_3}, \ldots, a_{i_k}a​i​1​​​​,a​i​2​​​​,a​i​3​​​​,…,a​i​k​​​​ thỏa mãn i1<i2<i3<…<iki_1 < i_2 < i_3 < \ldots < i_ki​1​​<i​2​​<i​3​​<…<i​k​​ và ai1<ai2<ai3<…<aika_{i_1} < a_{i_2} < a_{i_3} < \ldots < a_{i_k}a​i​1​​​​<a​i​2​​​​<a​i​3​​​​<…<a​i​k​​​​.

    Yêu cầu: Cho biết dãy con tăng đơn điệu dài nhất của dãy AAA đã cho có bao nhiêu phần tử.

    Input - Dữ liệu vào

    • Dòng đầu tiên là số nguyên dương nnn - số lượng phần tử của dãy AAA.
    • Dòng thứ hai gồm nnn số nguyên a1,a2,a3,…,ana_1, a_2, a_3, \ldots, a_na​1​​,a​2​​,a​3​​,…,a​n​​.

    Output - Dữ liệu ra

    • Độ dài của dãy con tăng đơn điệu dài nhất.

    Solution - Giải thuật

    • Gọi dp(i)dp(i)dp(i) là độ dài của dãy con tăng đơn điệu dài nhất nhận aia_ia​i​​ là phần tử cuối cùng của dãy.
    • Nhận xét: "Dãy có 111 phần tử luôn là dãy tăng đơn điệu". Do vậy, nếu mọi phần tử của dãy AAA là một dãy có độ dài 111 thì sẽ có được dp(i)=1∀i∈[1,n]dp(i) = 1 \forall i \in [1, n]dp(i)=1∀i∈[1,n].
    • Khi xét tới phần tử iii của dãy AAA, đặt dp(j)=m(j<i)dp(j) = m \; (j < i)dp(j)=m(j<i), khi đó, nếu aj<aia_j < a_ia​j​​<a​i​​ thì có thể nhận aia_ia​i​​ là phần tử cuối cùng của dãy con tăng đơn điệu dài nhất có độ dài m+1m + 1m+1. Nếu dp(i)<m+1dp(i) < m + 1dp(i)<m+1 thì cập nhật dp(i)=m+1dp(i) = m + 1dp(i)=m+1.

    C++:

    #include <bits/stdc++.h>

using namespace std;

int main() {
    int n;
    cin >> n;
    int *a = new int [n];
    for (int i = 0; i < n; ++i) {
        cin >> a[i];
    }

    int *dp = new int [n];
    for (int i = 0; i < n; ++i) {
        dp[i] = 1;
    }

    int ans = 1;
    for (int i = 1; i < n; ++i) {
        for (int j = 0; j < i; ++j) {
            if (a[j] < a[i]) {
                dp[i] = max(dp[i], dp[j] + 1);
            }
        }
        ans = max(ans, dp[i]);
    }

    cout << ans << endl;
    delete[] a;
    delete[] dp;
    return 0;
}
