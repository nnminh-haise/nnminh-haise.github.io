    Xếp hàng mua vé - NKTICK | Editorial - Lời giải

    Tags: Dynamic Programming, Basic

    Statement - Đề bài

    Nguồn: NKTICK | VNOI (opens new window)

    Có nnn người sắp hàng mua vé dự buổi hoà nhạc. Ta đánh số họ từ 111 đến nnn theo thứ tự đứng trong hàng. Mỗi người cần mua một vé, song người bán vé được phép bán cho mỗi người tối đa hai vé. Vì thế, một số người có thể rời hàng và nhờ người đứng trước mình mua hộ vé. Biết tit_it​i​​ là thời gian cần thiết để người iii mua xong vé cho mình. Nếu người i+1i + 1i+1 rời khỏi hàng và nhờ người iii mua hộ vé thì thời gian để người thứ iii mua được vé cho cả hai người là rir_ir​i​​.

    Yêu cầu: Xác định xem những người nào cần rời khỏi hàng và nhờ người đứng trước mua hộ vé để tổng thời gian phục vụ bán vé là nhỏ nhất.

    Input - Dữ liệu vào

    • Dòng đầu tiên là số nguyên dương n(1≤n≤60000)n \; (1 \le n \le 60000)n(1≤n≤60000).
    • Dòng thứ hai là nnn số nguyên dương t1,t2,t3,…,tn(1≤ti≤30000)t_1, t_2, t_3, \ldots, t_n \; (1 \le t_i \le 30000)t​1​​,t​2​​,t​3​​,…,t​n​​(1≤t​i​​≤30000).
    • Dòng thứ ba là n−1n - 1n−1 số nguyên dương r1,r2,r3,…,rn−1(1≤ri≤30000)r_1, r_2, r_3, \ldots, r_{n - 1} \; (1 \le r_i \le 30000)r​1​​,r​2​​,r​3​​,…,r​n−1​​(1≤r​i​​≤30000)

    Output - Dữ liệu ra

    In ra tổng thời gian phục vụ nhỏ nhất.

    Solution - Giải thuật

    • Gọi dp(i)dp(i)dp(i) là thời gian phục vụ nhỏ nhất từ người 111 đến người iii (người thứ iii là người mua vé).
    • Nếu dãy chỉ có 111 người thì dp(1)=t1dp(1) = t_1dp(1)=t​1​​.
    • Nếu dãy có 222 người thì dp(2)=min(t2+t1,r1)=min(t2+dp(1),r1)dp(2) = \min(t_2 + t_1, r_1) = min(t_2 + dp(1), r_1)dp(2)=min(t​2​​+t​1​​,r​1​​)=min(t​2​​+dp(1),r​1​​).
    • Khi dãy có iii người thì dp(i)=min(ti+dp(i−1),ri−1+dp(i−2))dp(i) = \min(t_i + dp(i - 1), r_{i - 1} + dp(i - 2))dp(i)=min(t​i​​+dp(i−1),r​i−1​​+dp(i−2)).

    C++:

    #include <bits/stdc++.h>

using namespace std;

int main() {
    int n;
    cin >> n;
    int* a = new int [n];
    int* b = new int [n - 1];
    for (int i = 0; i < n; ++i) {
        cin >> a[i];
    }
    for (int i = 0; i < n - 1; ++i) {
        cin >> b[i];
    }

    long long* dp = new long long [n];
    dp[0] = a[0];
    dp[1] = min(a[1] + a[0], b[0]);
    for (int i = 2; i < n; ++i) {
        dp[i] = min(a[i] + dp[i - 1], b[i - 1] + dp[i - 2]);
    }

    cout << dp[n - 1] << endl;
    return 0;
}
