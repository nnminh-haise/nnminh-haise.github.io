    Tags: Math, Modulo, Basic

    Statement - Đề bài

    Cho một dãy AAA gồm nnn số nguyên dương a1,a2,a3,…ana_1, a_2, a_3, \ldots a_na​1​​,a​2​​,a​3​​,…a​n​​. Với mỗi số aia_ia​i​​ hãy kiểm tra số đó chia hết cho 777 hoặc 131313 hay không.

    • Nếu aia_ia​i​​ chia hết cho 777 in mod 7.
    • Nếu aia_ia​i​​ chia hết cho 131313 in mod 13.
    • Nếu aia_ia​i​​ chia hết cho cả 777 và 131313 in both.
    • Nếu aia_ia​i​​ không chia hết cho 777 hoặc 131313 in none.

    Yêu cầu: Với mỗi phần tử aia_ia​i​​ hãy in ra câu trả lời tương ứng.

    Input - Dữ liệu vào

    • Dòng đầu tiên là số nguyên dương n(1≤n≤1000)n \; (1 \le n \le 1000)n(1≤n≤1000).
    • Dòng thứ hai gồm nnn số nguyên a1,a2,a3,…,an(1≤ai≤1000)a_1, a_2, a_3, \ldots, a_n \; (1 \le a_i \le 1000)a​1​​,a​2​​,a​3​​,…,a​n​​(1≤a​i​​≤1000).

    Output - Dữ liệu ra

    • Gồm nnn dòng, mỗi dòng là đầu ra tương ứng với phần tử aia_ia​i​​.

    Solution - Giải thuật

    Ta sẽ duyệt tuần tự qua các phần tử trong dãy đã cho và kiểm tra từng điều kiện mà đề đã yêu cầu.

    C++:

    #include <iostream>

using namespace std;

int main() {
    int n;
    cin >> n;
    for (int i = 1; i <= n; ++i) {
        int val;
        cin >> val;
        if (val % 7 == 0 && val % 13 == 0) {
            cout << "both\n";
        }
        else if (val % 7 == 0) {
            cout << "mod 7\n";
        }
        else if (val % 13 == 0) {
            cout << "mod 13\n";
        }
        else {
            cout << "none\n";
        }
    }
    return 0;
}
