    Tags: Basic, Recursion

    Statement - Đề bài

    Cho số nguyên dương nnn.

    Yêu cầu: Thiết kế một hàm đệ quy tính giá trị sau.

    S=1+12+13+14+…+1nS = 1 + \dfrac{1}{2} + \dfrac{1}{3} + \dfrac{1}{4} + \ldots + \dfrac{1}{n} S=1+​2​​1​​+​3​​1​​+​4​​1​​+…+​n​​1​​

    Input - Dữ liệu vào

    • Một số nguyên dương nnn duy nhất.

    Output - Dữ liệu ra

    • Một số thực SSS duy nhất là kết quả của phép tính.

    Solution - Giải thuật

    Gọi hàm f(n)f(n)f(n) là hàm đệ quy tính tổng SSS tại nnn. Như mọi hàm đệ quy khác thì cần phải xác định điểm neo và công thức truy hồi cho hàm.

    • Điểm neo: Dựa trên công thức, số hạng đầu tiên của tổng trên luôn là 111. Do đó, có thể chọn n=1n = 1n=1 là điểm neo của hàm số.

    • Công thức truy hồi: Giả sử, xét hàm số f(n)f(n)f(n), giá trị f(n)=1n+f(n−1)f(n) = \dfrac{1}{n} + f(n - 1)f(n)=​n​​1​​+f(n−1) vì số hạng thứ nnn sẽ là 1n\dfrac{1}{n}​n​​1​​ và S=f(n)=1n+1n−1+1n−2+…+1=1n+f(n−1)S = f(n) = \dfrac{1}{n} + \dfrac{1}{n - 1} + \dfrac{1}{n - 2} + \ldots + 1 = \dfrac{1}{n} + f(n - 1)S=f(n)=​n​​1​​+​n−1​​1​​+​n−2​​1​​+…+1=​n​​1​​+f(n−1).

    Python:

    def f(n: float) -> float:
    if n == 1:
        return 1
    
    return (1 / n + f(n - 1))


n = float(input())
print(f(n))
