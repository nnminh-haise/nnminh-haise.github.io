    Dãy nhị phân | Editorial - Lời giải

    Tags: Recursion, Basic

    Statement - Đề bài

    Một dãy gồm nnn phần tử chỉ bao gồm các số 000 hoặc 111 được gọi là dãy nhị phân độ dài nnn.

    Yêu cầu: Cho số nguyên dương nnn. In ra tất cả các dãy nhị phân độ dài nnn được sắp xếp theo thứ tự từ điển.

    Input - Dữ liệu vào

    • Một số nguyên dương nnn duy nhất.

    Output - Dữ liệu ra

    • Gồm nhiều dòng, mỗi dòng là một dãy nhị phân độ dài nnn. Mỗi phần tử của dãy được ngăn cách bởi dấu cách.

    Solution - Giải thuật

    C++:

    class Solution {
public:
    void binaryArray(const int n, int index, int *x) {
        if (index == n) {
            for (int i = 0; i < n; ++i) {
                cout << x[i] << (i < n - 1 ? " " : "\n");
            }
            return;
        }
        for (int i = 0; i <= 1; ++i) {
            x[index] = i;
            recursion(n, index + 1, x);
        }
    }
};
