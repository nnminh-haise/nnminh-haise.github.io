    GCD & LCM | Editorial - Lời giải

    Tags: Modulo, GCD, Math, Basic

    Statement - Đề bài

    Cho hai số nguyên dương aaa và bbb.

    Yêu cầu: Tìm ước chung lớn nhất và bội chung nhỏ nhất của aaa và bbb.

    Input - Dữ liệu vào

    • Dòng đầu tiên là số nguyên dương q(1≤q≤1000)q \; (1 \le q \le 1000)q(1≤q≤1000) - số truy vấn cần trả lời. Mỗi truy vấn sẽ gồm hai số nguyên dương aia_ia​i​​ và bib_ib​i​​.
    • qqq dòng tiếp theo, mỗi dòng là hai số ai,bi(1≤ai,bi≤106)a_i, b_i \; (1 \le a_i, b_i \le 10^6)a​i​​,b​i​​(1≤a​i​​,b​i​​≤10​6​​) ứng với mỗi truy vấn.

    Output - Dữ liệu ra

    • Gồm qqq dòng, mỗi dòng gồm hai số lần lượt là gcd(ai,bi)\gcd(a_i, b_i)gcd(a​i​​,b​i​​) và lcm(ai,bi)\text{lcm}(a_i, b_i)lcm(a​i​​,b​i​​) tương ứng với truy vấn.

    Solution - Giải thuật

    Đây là bài toán cơ bản áp dụng giải thuật tìm ước chung lớn nhất và bội chung nhỏ nhất của hai số nguyên dương. Lưu ý về độ lớn của bội chung nhỏ nhất khi thực hiện tính toán.

    C++:

    #include <iostream>

using namespace std;

long long gcd(long long a, long long b) {
    return (b ? gcd(b, a % b) : a);
}

long long lcm(long long a, long long b) {
    return a * b / gcd(a, b);
}

int main() {
	int queries;
	cin >> queries;
	while (queries--) {
        long long a, b;
        cin >> a >> b;
        cout << gcd(a, b) << " " << lcm(a, b) << endl;
	}
	return 0;
}

    def gcd(a, b):
    if b == 0:
        return a

    return gcd(b, a % b)

def lcm(a, b):
    return a * b // gcd(a, b)


queries = int(input())

while queries:
    queries -= 1
    
    a, b = list(map(int, input().split(' ')))
    print(gcd(a, b), lcm(a, b))
