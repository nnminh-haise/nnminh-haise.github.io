Luyện tập | Editorial - Lời giải

Tags: Modulo, GCD, LCM, Math, Basic

Statement - Đề bài

Kì thi học kì sắp diễn ra nên Tùng và Nhi quyết định lên kế hoạch ôn tập để đạt được kết quả thật cao. Có nnn bài tập để Tùng và Nhi có thể cùng nhau ôn luyện được đánh số từ 111 đến nnn. Tùng và Nhi quyết định phải làm tối thiểu kkk bài tập thì mới được nghỉ ngơi. Nhưng Tùng chỉ làm những bài có chỉ số chia hết cho aaa còn Nhi chỉ làm những bài có chỉ số chia hết cho bbb và sẽ bỏ qua những bài có chỉ số cùng chia hết cho cả a,ba, ba,b.

Yêu cầu: Đếm số bài tập mà Tùng và Nhi sẽ làm và xác định liệu cả hai có hoàn thành mục tiêu làm tối thiểu kkk bài tập đã đề ra hay không.

Input - Dữ liệu vào

  • Dòng đầu tiên là số nguyên dương q(1≤q≤10)q \; (1 \le q \le 10)q(1≤q≤10) - số truy vấn cần trả lời. Mỗi truy vấn có dạng như sau:

    • Mỗi truy vấn gồm 444 số nguyên dương n,a,b,k(1≤n,k≤1018,1≤a,b≤106)n, a, b, k \; (1 \le n, k \le 10^{18}, 1 \le a, b \le 10^6)n,a,b,k(1≤n,k≤10​18​​,1≤a,b≤10​6​​) theo thứ tự như đã mô tả.

Output - Dữ liệu ra

  • Gồm qqq dòng, mỗi dòng là câu trả lời tương ứng với truy vấn nếu Tùng và Nhi có thể làm được tối thiểu kkk bài tập in Yes, ngược lại in No.

Subtasks

  • Subtask 111 (50%50\%50% số điểm) thỏa mãn:

    • 1≤n,k≤1061 \le n, k \le 10^61≤n,k≤10​6​​.
    • 1≤a,b≤1031 \le a, b \le 10^31≤a,b≤10​3​​.

  • Subtask 222 (50%50\%50% số điểm còn lại) không có yêu cầu gì thêm.

Solution - Giải thuật

Naive solution - Giải thuật ngây thơ

Đối với Subtask 111, ta có thể dễ dàng sử dụng một vòng for từ 111 đến nnn để đếm những số chia hết cho aaa hoặc chia hết cho bbb.

Độ phức tạp thời gian: O(n)O(n)O(n).

C++:

#include <iostream>

using namespace std;

unsigned long long queries = 0, n = 0, k = 0, ans = 0, a = 0, b = 0;

int main() {
	cin >> queries;
	
	while (queries--) {
        cin >> n >> a >> b >> k;
        ans = 0;
        
        for (int i = 1; i <= n; ++i) {
            if ((a % i == 0 && b % i != 0) || (a % i != 0 && b % i == 0)) {
                ++ans;
            }
        }

        if (ans >= k) {
            cout << "Yes\n";
        }
        else {
            cout << "No\n";
        }
	}
	return 0;
}

queries = int(input())

Optimized solution - Giải thuật tối ưu

Đối với subtask 222, ta không thể dùng vòng lặp for đối với subtask 111 vì giá trị tối đa của nnn có thể lên đến 101810^{18}10​18​​. Do vậy ta cần phải nghĩ ra một cách khác để giải quyết bài toán.

Ta gọi AAA là tập các số chia hết cho aaa trong đoạn từ 111 đến nnn và BBB là tập các số chia hết cho bbb trong đoạn từ 111 đến nnn. Với cách định nghĩa như vậy thì tập các phần tử chia hết cho cả aaa và bbb trong đoạn từ 111 đến nnn sẽ được xác định bằng: C=A∩BC = A \cap BC=A∩B.

Ví dụ: với a=2,b=4,n=10a = 2, b = 4, n = 10a=2,b=4,n=10 ta có: A={2,4,6,8,10},B={4,8}A = \{2, 4, 6, 8, 10\}, \; B = \{4, 8\}A={2,4,6,8,10},B={4,8} và C=A∩B={4,8}C = A \cap B = \{4, 8\}C=A∩B={4,8}. Lúc này, kết quả bài toán sẽ là số lượng các phần tử trong AAA và BBB nhưng không thuộc tập CCC.

S=(A∪B)∖(A∩B)S = (A \cup B) \setminus (A \cap B) S=(A∪B)∖(A∩B)

Ta có thể đếm số lượng phần tử của tập AAA bằng công thức: ∣A∣=⌊na⌋|A| = \left\lfloor \dfrac{n}{a} \right\rfloor∣A∣=⌊​a​​n​​⌋. Áp dụng tương tự cho BBB và CCC ta được ∣B∣=⌊nb⌋|B| = \left\lfloor \dfrac{n}{b} \right\rfloor∣B∣=⌊​b​​n​​⌋ và ∣C∣=⌊na⋅b⌋|C| = \left\lfloor \dfrac{n}{a \cdot b} \right\rfloor∣C∣=⌊​a⋅b​​n​​⌋. Vậy số lượng phần tử của tập SSS sẽ được xác định bằng cong thức:

∣S∣=∣A∣+∣B∣−2×∣C∣|S| = |A| + |B| - 2 \times |C| ∣S∣=∣A∣+∣B∣−2×∣C∣

Độ phức tạp thời gian: O(1)O(1)O(1).

C++:

#include <iostream>
#include <math.h>

using namespace std;

unsigned long long queries = 0, n = 0, k = 0, ans = 0, a = 0, b = 0;

unsigned long long gcd(unsigned long long a, unsigned long long b) {
    return b ? gcd(b, a % b) : a;
}

unsigned long long lcm(unsigned long long a, unsigned long long b) {
    return a / gcd(a, b) * b;
}

int main() {
	cin >> queries;
	
	while (queries--) {
        cin >> n >> a >> b >> k;
        ans = 0;
        
        ans = floor(n / a) + floor(n / b) - 2 * floor(n / lcm(a, b));

        if (ans >= k) {
            cout << "Yes\n";
        }
        else {
            cout << "No\n";
        }
	}
	return 0;
}

Python:

import math

def gcd(a, b):
    if b == 0:
        return a

    return gcd(b, a % b)

def lcm(a, b):
    return a // gcd(a, b) * b


queries = int(input())

while queries:
    queries -= 1
    
    n, a, b, k = list(map(int, input().split(' ')))

    ans = math.floor(n // a) + math.floor(n // b) - 2 * math.floor(n // lcm(a, b))

    if ans >= k:
        print("Yes")
    else:
        print("No")
