    Tags: Searching, Binary Search, Sorting

    Statement - Đề bài

    Cho dãy AAA có n(1≤n≤10000)n \; (1 \le n \le 10000)n(1≤n≤10000) phần tử a1,a2,a3,…,an(1≤ai≤10000)a_1, a_2, a_3, \ldots, a_n \; (1 \le a_i \le 10000)a​1​​,a​2​​,a​3​​,…,a​n​​(1≤a​i​​≤10000) là các số nguyên dương và một số nguyên dương k(1≤k≤10000)k \; (1 \le k \le 10000)k(1≤k≤10000). Dãy AAA được đánh chỉ số từ 000.

    Cận dưới của một số kkk bất kì trong dãy AAA là vị trí nhỏ nhất của kkk trong dãy AAA đã sắp xếp hoặc vị trí lớn nhất của số nguyên lớn nhất nhỏ hơn kkk.

    Yêu cầu: Tìm cận dưới của kkk trong dãy AAA.

    Input - Dữ liệu vào

    • Dòng đầu tiên gồm hai số nguyên dương nnn và kkk.
    • Dòng thứ hai gồm nnn số nguyên dương a1,a2,a3,…,ana_1, a_2, a_3, \ldots, a_na​1​​,a​2​​,a​3​​,…,a​n​​.

    Output - Dữ liệu ra

    • Một số nguyên duy nhất là chỉ số của cận dưới của kkk trong dãy AAA.

    Solution - Giải thuật

    Đây là một cải tiến của thuật tìm kiếm nhị phân cơ bản.

    Sau khi sắp xếp dãy theo chiều tăng dần. Gọi lll và rrr là chỉ số đầu và chỉ số cuối của dãy. Khởi tạo p=+∞p = +\inftyp=+∞ là chỉ số của cận dưới.

    • Chừng nào l≤rl \le rl≤r thì lặp lại các bước sau:

      • mid=(r−l+1)÷2+lmid = (r - l + 1) \div 2 + lmid=(r−l+1)÷2+l
      • Nếu k≤a[mid]k \le a[mid]k≤a[mid] thì cập nhật p=min(p,mid)p = \min(p, mid)p=min(p,mid) và r=mid−1r = mid - 1r=mid−1.
      • Ngược lại, l=mid+1l = mid + 1l=mid+1.

    • Khi vòng lặp kết thúc, giá trị của ppp là chỉ số của cận dưới của kkk trong dãy AAA.

    C++:

    #include <bits/stdc++.h>

using namespace std;

int lowerBound(vector < int > &a, int k) {
    int p = 1e9 + 7;
    int left = 0, right = a.size() - 1;
    while (left <= right) {
        int mid = (right - left + 1) / 2 + left;
        if (k <= a[mid]) {
            p = min(p, mid);
            right = mid - 1;
        }
        else {
            left = mid + 1;
        }
    }
    return p;
}

int main() {
    int n, k;
    cin >> n >> k;
    vector < int > a(n);
    for (int &item: a) {
        cin >> item;
    }

    cout << lowerBound(a, k) << endl;
    return 0;
}
