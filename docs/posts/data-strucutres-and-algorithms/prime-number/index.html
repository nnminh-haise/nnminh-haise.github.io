        Prime number & Sieve of Eratosthenes - Số nguyên tố & Thuật toán sàng nguyên tố Eratosthens

        Đố cậu! Số nguyên tố là gì? Làm sao để tìm một số lượng lớn các số nguyên tố trong khoảng?

        Prime number - Số nguyên tố

        Definition - Định nghĩa

        Số nguyên tố là số tự nhiên lớn hơn 111 có duy nhất hai ước số là 111 và chính nó. Các số không phải số nguyên tố được gọi là hợp số.

        Ví dụ: Số 555 là số nguyên tố vì số 555 có duy nhất hai ước số là 111 và 555. Còn số 101010 không phải là số nguyên tố vì 101010 có nhiều hơn 222 ước số: 1,2,5,101, 2, 5, 101,2,5,10.

        Primality test - Thuật toán kiểm tra số nguyên tố

        Nếu một số là số nguyên tố thì nó được gọi là có tính nguyên tố (primality).

        Naive Algorithm - Thuật toán ngây thơ

        Dựa vào định nghĩa của số nguyên tố, cậu có thể kiểm tra một số nguyên tố bằng cách đếm số lượng ước số của nó. Nếu số lượng ước số đo đúng bằng hai ước thì số đó chính là số nguyên tố.

        C++:

        bool is_prime(int x) {
    if (x <= 1) {
        return false;
    }

    int counter = 0;

    for (int i = 1; i <= x; ++i) {
        if (x % i == 0) {
            ++counter;
        }
    }

    if (counter == 2) {
        return true;
    }
    return false;
}

        Optimized Algorithm - Thuật toán tối ưu

        Để cải tiến thuật toán trên cậu cần có một số nhận xét sau:

        • Số nguyên tố chẵn duy nhất là số 222 và cũng là số nguyên tố đầu tiên.
        • Các số nguyên tố, trừ số 222, đều là số lẻ.
        • Nếu giả sử ban đầu số xxx là một số nguyên tố, nếu tồn tại một số i∈[3,x]i \in [3, \sqrt{x}]i∈[3,√​x​​​] là ước của xxx thì điều đó chứng tỏ giả định ban đầu là sai, hay xxx không phải là số nguyên tố.

        Từ những nhận xét trên, cậu có một thuật toán kiểm tra tính nguyên tố tối ưu như sau:

        Sieve of Eratosthenes - Thuật toán sàng số nguyên tố Eratosthens

        Đặt vấn đề: In ra tất cả các số nguyên tố trong đoạn [1,106][1, 10^6][1,10​6​​].

        Naive Algorithm - Giải thuật ngây thơ

        Cậu có thể áp dụng ngay thuật toán kiểm tra tính nguyên tố của một số nguyên bất kì ở trên để giải quyết bài toán trên. Cụ thể, cậu sẽ sử dụng vòng lặp for để duyệt qua các giá trị iii trong đoạn [1,106][1, 10^6][1,10​6​​], với mỗi giá trị iii cậu sẽ kiểm tra iii có phải là số nguyên tố hay không, nếu phải thì in ra ngoài màn hình.

        C++:

        #include <iostream>
#include <math.h>

using namespace std;

bool is_prime(int x) {
    if (x < 2) {
        return false;
    }
    if (x == 2 || x == 3) {
        return true;
    }
    if (x % 2 == 0) {
        return false;
    }
    for (int i = 3; i <= sqrt(x); i += 2) {
        if (x % i == 0) {
            return false;
        }
    }
    return true;
}

int main() {
    for (int i = 1; i <= (int)1e6; ++i) {
        if (is_prime(i) == true) {
            cout << i << " ";
        }
    }
    cout << endl;

    return 0;
}

        Sieve of Eratosthenes - Thuật toán sàng số nguyên tố

        Thuật toán sàng số nguyên tố là thuật toán cho phép ta tìm tất cả các số nguyên tố trong đoạn [1,n][1, n][1,n] với độ phức tạp thời gian là O(nloglogn)O(n \log \log n)O(nloglogn).

        Ý tưởng cơ bản của thuật toán này có thể được mô tả đơn giản như sau: nếu số ppp là số nguyên tố thì các số 2p,3p,…,kp≤n2p, 3p, \ldots, kp \le n2p,3p,…,kp≤n chắc chắn không phải số nguyên tố. Do đó ta sẽ đánh dấu các số này là các hợp số. Tương tự với các số nguyên tố ppp tiếp theo cho đến khi tất cả các số trong đoạn [1,n][1, n][1,n] được đánh dấu hết.

        Dễ thấy có vài số sẽ bị đánh dấu nhiều hơn 111 lần, nhưng số lượng các số này không nhiều khi nnn lớn nên ta có thể bỏ qua.

        C++:

        void sieveOfEratosthenes() {
    int n = (int)1e6;
    vector < bool > is_prime(n + 1, true);
    is_prime[0] = is_prime[1] = false;
    for (int p = 2; p <= n; ++p) {
        if (is_prime[p] == true && (long long)p * p <= n) {
            for (int i = p * p; i <= n; i += p) {
                is_prime[i] = false;
            }
        }
    }
    for (int i = 1; i <= n; ++i) {
        if (is_prime[i] == true) {
            cout << i << " ";
        }
    }
    cout << endl;
}

        Python:

        def sieveOfEratosthenes():
    n = 1000000
    is_prime = [True for i in range(0, n + 1)]
    
    is_prime[0] = is_prime[1] = False
    
    for p in range(2, n + 1):
        if is_prime[p] == True and p * p <= n:
            for i in range(p * p, n + 1, p):
                is_prime[i] = False
    
    for i in range(1, n + 1):
        if is_prime[i] == True:
            print(i)
