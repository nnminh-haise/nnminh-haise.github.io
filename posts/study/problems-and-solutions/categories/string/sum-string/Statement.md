
# Cộng chuỗi

Cho hai chuỗi $s_1$ và $s_2$ có độ dài không quá $1000$ phần tử.

***Yêu cầu:*** đưa ra chuỗi $s = s_1 + s_2$.

## Input - Dữ liệu vào

- Gồm hai dòng, dòng đầu tiên là chuỗi $s_1$ và dòng thứ hai là chuỗi $s_2$.

## Output - Dữ liệu ra

- Một dòng duy nhất là chuỗi $s$.

## Sample - Ví dụ

#### Input:

```
abc
def
```

#### Output:

```
abcdef
```
